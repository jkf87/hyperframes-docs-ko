<Note>


  키보드 단축키는 플랫폼 및 터미널에 따라 다를 수 있습니다. 환경에 사용 가능한 단축키를 보려면 `?`를 누르세요.

  **macOS 사용자**: Option/Alt 키 단축키(`Alt+B`, `Alt+F`, `Alt+Y`, `Alt+M`, `Alt+P`, `Alt+T`)를 사용하려면 터미널에서 Option을 Meta로 구성해야 합니다:

  * **iTerm2**: settings → Profiles → Keys → Left/Right Option key를 "Esc+"로 설정
  * **Terminal.app**: settings → Profiles → Keyboard → "Use Option as Meta Key" 확인란 선택
  * **VS Code**: VS Code 설정에서 `"terminal.integrated.macOptionIsMeta": true` 설정

  자세한 내용은 [터미널 구성](/terminal-config)을 참조하세요.
</Note>
