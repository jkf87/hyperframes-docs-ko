# 빠른 시작

> 2분 안에 첫 번째 Hyperframes 영상을 만들고, 미리 보고, 렌더링하세요.

AI 에이전트에게 프롬프트를 입력하거나 프로젝트를 직접 시작하여, 아무것도 없는 상태에서 렌더링된 MP4까지 한번에 완성할 수 있습니다.

## 옵션 1: AI 코딩 에이전트 사용 (권장)

HyperFrames 스킬을 설치한 다음, 원하는 영상을 설명하세요:

```bash
npx skills add heygen-com/hyperframes
```

이 명령은 에이전트(Claude Code, Cursor, Gemini CLI, Codex)에게 올바른 컴포지션과 GSAP 애니메이션 작성법을 학습시킵니다. Claude Code에서는 슬래시 커맨드로 등록됩니다. `/hyperframes`는 컴포지션 작성, `/hyperframes-cli`는 CLI 명령, `/gsap`는 애니메이션 관련 도움을 위한 것입니다. 슬래시 커맨드를 호출하면 스킬 컨텍스트가 명시적으로 로드되어 처음부터 정확한 결과물을 생성합니다.

### 직접 해보기: 예시 프롬프트

- **콜드 스타트 — 원하는 것을 설명하기**
  - `/hyperframes`를 사용해서, 어두운 배경 위에 페이드인 타이틀과 은은한 배경 음악이 있는 10초짜리 제품 소개 영상을 만들어 줘.
- **웜 스타트 — 기존 컨텍스트를 영상으로 변환하기**
  - 이 GitHub 저장소 https://github.com/heygen-com/hyperframes 를 살펴보고, `/hyperframes`를 사용해서 용도와 아키텍처를 설명하는 영상을 만들어 줘.
  - 첨부된 PDF를 `/hyperframes`를 사용해서 45초짜리 피치 영상으로 요약해 줘.
  - 이 CSV를 `/hyperframes`를 사용해서 애니메이션 막대 차트 레이스로 만들어 줘.
- **포맷별 요청**
  - `/hyperframes`를 사용해서 [주제]에 대한 9:16 TikTok 스타일 훅 영상을 만들어 줘. TTS 내레이션에 맞춰 동기화된 통통 튀는 자막도 넣어 줘.
- **반복 수정 — 에이전트를 영상 편집자처럼 대하기**
  - 타이틀을 2배 크게 하고, 다크 모드로 바꾸고, 끝에 페이드아웃을 추가해 줘.
  - 0:03 지점에 내 이름과 직함이 들어간 로워 서드를 추가해 줘.

에이전트가 스캐폴딩, 애니메이션, 렌더링을 모두 처리합니다. 더 많은 패턴은 프롬프팅 가이드를 참고하세요.

> 스킬은 일반 웹 문서에는 없는 HyperFrames 고유 패턴을 담고 있습니다. 예를 들어 시간 기반 요소에 필요한 `class="clip"`, GSAP 타임라인 등록, `data-*` 속성 시맨틱스 등이 이에 해당합니다. 스킬을 사용하면 처음부터 올바른 컴포지션을 생성할 수 있습니다.
