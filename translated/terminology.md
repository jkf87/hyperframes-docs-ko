# 용어집 (Terminology)

Hyperframes 문서 한국어 번역에서 사용한 주요 용어 표준화 목록입니다.

| 영문 원문 | 한국어 번역 | 비고 |
| --- | --- | --- |
| composition | 컴포지션 | 영상의 구성 단위, 음차 표기 |
| clip | 클립 | 타이밍이 지정된 개별 요소 |
| render / rendering | 렌더링 | 영상 출력 과정 |
| timeline | 타임라인 | GSAP 애니메이션 타임라인 |
| track | 트랙 | 클립이 배치되는 레이어 |
| scaffold | 스캐폴드 | 프로젝트 초기 구조 생성 |
| preview | 미리보기 | 브라우저 실시간 확인 |
| dev server | 개발 서버 | 로컬 미리보기 서버 |
| hot reload | 핫 리로드 | 파일 변경 시 자동 반영 |
| frame | 프레임 | 영상의 개별 장면 단위 |
| encoding | 인코딩 | 영상 압축·변환 과정 |
| overlay | 오버레이 | 화면 위에 겹쳐 표시되는 요소 |
| transition | 전환 효과 | 장면 간 전환 |
| fade | 페이드 | 투명도 기반 전환 |
| slide | 슬라이드 | 이동 기반 전환 |
| scale | 스케일 | 크기 변화 전환 |
| data visualization | 데이터 시각화 | — |
| preset | 프리셋 | 미리 정의된 설정 |
| deterministic render | 결정적 렌더링 | 동일 입력 → 동일 출력 보장 |
| root element | 루트 요소 | 최상위 DOM 요소 |
| package manager | 패키지 매니저 | npm, bun 등 |
| AI coding agent | AI 코딩 에이전트 | Claude Code, Cursor 등 |
